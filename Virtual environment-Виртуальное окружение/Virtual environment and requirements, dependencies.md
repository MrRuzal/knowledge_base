Просмотр установленных зависимостей 
```
pip freeze   
```
Запись  **requirements** 
```
pip freeze > requirements.txt   
```
Установка **requirements ** библиотек 
```
pip install -r requirements.txt  
``` 
**Активация виртуального окружения (с определенной версией python)**
```
py -3.11 -m venv venv
```
